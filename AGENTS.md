# AGENTS.md

## Project Rules

- Keep the site professional, responsive, accessible, and optimized for software engineering job applications.
- Prioritize engineering work and experience over academics, honors, or general biography.
- Do not let the site drift into a long resume page or an awards archive.
- Keep the main background light and mostly white.
- Use the pastel palette selectively for cards, borders, buttons, tags, hover states, and subtle decorative accents.
- Centralize all portfolio content in `src/data/portfolio.ts`.
- Use reusable, data-driven components instead of scattering content across page files.
- Use explicit placeholders for any missing dates, metrics, technologies, URLs, media, or descriptions.
- External links must open safely in a new tab.
- After meaningful changes, run `npm run build` and `npm run typecheck`.
